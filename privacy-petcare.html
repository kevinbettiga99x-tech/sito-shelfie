Informativa sulla Privacy

PetCare Assistant è un'applicazione mobile progettata pensando alla massima privacy dell'utente. Non richiede account, non raccoglie dati personali e non utilizza alcun server proprietario.

Questa policy descrive come vengono gestiti i dati all'interno dell'app. L'utilizzo di PetCare Assistant implica l'accettazione delle pratiche descritte qui di seguito.

Dati memorizzati localmente

Tutti i dati inseriti nell'app — inclusi profili degli animali, eventi sanitari, promemoria, diario, pesi, attività, spese, scorte e documenti — sono salvati esclusivamente nella memoria locale del dispositivo tramite un database SQLite criptato. Lo sviluppatore non ha accesso a queste informazioni in nessun caso.

Permessi richiesti

L'app richiede i seguenti permessi di sistema, ognuno utilizzato esclusivamente per la funzione indicata:

  • Fotocamera — per scansionare le etichette alimentari dei prodotti (OCR on-device). Nessuna immagine viene trasmessa all'esterno.
  • Posizione (in uso) — per ricevere avvisi meteo personalizzati nella sezione Avvisi Ambientali. La posizione non viene mai memorizzata né trasmessa a server propri. Viene inviata anonimamente all'API pubblica Open-Meteo (open-meteo.com) solo quando si apre la sezione meteo.
  • Memoria / file — per importare file CSV di prodotti e per la funzione di backup/ripristino locale.
  • Notifiche — per inviare promemoria locali (visite, farmaci, scadenze). Nessuna notifica è generata da server remoti.
  • Allarmi esatti — necessari per garantire che i promemoria medici (vaccinazioni, farmaci) vengano inviati all'orario preciso impostato dall'utente.

Connessioni a internet

PetCare Assistant effettua connessioni a internet solo per due funzioni opzionali, entrambe con API pubbliche gratuite che non richiedono autenticazione né raccolgono dati personali:

  • Avvisi Ambientali / Meteo — tramite Open-Meteo (open-meteo.com): API meteo pubblica, senza account, senza tracciamento. Vengono inviate solo le coordinate geografiche del momento, non memorizzate.
  • Ricerca Marche — tramite Open Pet Food Facts (world.openpetfoodfacts.org): database aperto di prodotti per animali. La ricerca è anonima e non associata a nessun account utente.

Tutte le altre funzioni dell'app — diario, salute, promemoria, alimentazione, allenamento, emergenze — funzionano completamente offline.

Condivisione dei dati

PetCare Assistant non condivide, vende né trasmette dati personali a terze parti. Non vengono utilizzati strumenti di analytics, pubblicità, tracciamento comportamentale o SDK di social network.

Backup e ripristino

L'app include una funzione di backup manuale che genera un file .petcare contenente tutti i dati dell'utente. Questo file viene salvato o condiviso esclusivamente tramite le app scelte dall'utente (Drive, email, ecc.) e non viene inviato ad alcun server dello sviluppatore.

Sicurezza

Il database locale utilizza SQLite. Poiché nessun dato viene trasmesso a server propri, il rischio di violazioni della privacy è strutturalmente nullo. La sicurezza dei dati dipende dal blocco schermo e dalle impostazioni di sicurezza del dispositivo dell'utente.

Avvertenza medica

PetCare Assistant è uno strumento di supporto organizzativo, non uno strumento medico-veterinario. Le informazioni presenti nell'app (sintomi, guide, consigli nutrizionali) hanno scopo puramente informativo. Per qualsiasi problema di salute del tuo animale, consulta sempre un veterinario qualificato.

Privacy Policy

PetCare Assistant is a mobile application designed with user privacy as a top priority. No account is required, no personal data is collected, and no proprietary server infrastructure is used.

This policy describes how data is handled within the app. Using PetCare Assistant implies acceptance of the practices described below.

Locally Stored Data

All data entered in the app — including pet profiles, health events, reminders, diary entries, weights, activities, expenses, stock items and documents — is stored exclusively on the device's local memory using a local SQLite database. The developer has no access to this information under any circumstances.

Permissions

The app requests the following system permissions, each used solely for the stated purpose:

  • Camera — to scan pet food labels using on-device OCR. No images are transmitted externally.
  • Location (while in use) — to provide personalised weather alerts in the Environmental Alerts section. Location is never stored and is sent anonymously to the public Open-Meteo API (open-meteo.com) only when the weather section is opened.
  • Storage / files — to import CSV product files and for the local backup/restore feature.
  • Notifications — to send local reminders (vet visits, medications, deadlines). No notifications are generated by remote servers.
  • Exact alarms — required to ensure medical reminders (vaccinations, medications) are delivered at the exact time set by the user.

Internet Connections

PetCare Assistant connects to the internet only for two optional features, both using free public APIs that require no authentication and collect no personal data:

  • Environmental Alerts / Weather — via Open-Meteo (open-meteo.com): a public weather API with no account requirement and no tracking. Only the current coordinates are sent, and they are not stored.
  • Brand Search — via Open Pet Food Facts (world.openpetfoodfacts.org): an open pet food product database. Searches are anonymous and not linked to any user account.

All other app features — diary, health, reminders, nutrition, training, emergencies — work completely offline.

Data Sharing

PetCare Assistant does not share, sell or transmit personal data to any third party. No analytics tools, advertising SDKs, behavioural tracking or social media SDKs are used.

Backup & Restore

The app includes a manual backup feature that generates a .petcare file containing all user data. This file is saved or shared exclusively through apps chosen by the user (Drive, email, etc.) and is not sent to any developer-controlled server.

Security

The local database uses SQLite. Since no data is transmitted to proprietary servers, the risk of privacy breaches is structurally zero. Data security depends on the user's device screen lock and security settings.

Medical Disclaimer

PetCare Assistant is an organisational support tool, not a medical or veterinary instrument. Information within the app (symptoms, guides, nutritional advice) is provided for informational purposes only. For any health concern relating to your pet, always consult a qualified veterinarian.
